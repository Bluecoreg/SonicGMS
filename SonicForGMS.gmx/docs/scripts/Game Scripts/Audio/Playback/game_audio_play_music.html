game_audio_play_music

Syntax: 

game_audio_play_music(soundid, priority, loops);
Argument name Type Description
soundid real sound index
priority real (optional) channel priority; lower priority sounds may be cut off
loops boolean (optional) whether or not the music repeats indefinitely

Returns: (real) index of played audio (-1 on error)

Description:

Starts music playback (Stops all previous instances of the sound before playing)
