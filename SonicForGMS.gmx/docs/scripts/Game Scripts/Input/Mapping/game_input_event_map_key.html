game_input_event_map_key

Syntax: 

game_input_event_map_key(event, key);
Argument name Type Description
event string event name
key real keyboard key (see GameMaker documentation for more info)

Description:

Binds the given keyboard key to the given event. (Keyboard events are always registered on input channel 0)
