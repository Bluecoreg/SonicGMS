game_sign_add_moment

Syntax: 

game_sign_add_moment(ind, duration, delay, xpos, ypos, xscale, yscale, rot, color, alpha);
Argument name Type Description
ind real instance index of AnimatedGameSign
duration real amount of frames to animate
delay real amount of frames to wait before animating
xpos real x-position to move towards
ypos real y-position to move towards
xscale real horizontal scale to stretch the image towards
yscale real vertical scale to stretch the image towards
rot real angle to rotate towards
color real color to blend towards
alpha real transparency level to blend towards

Description:

Adds a new animation to an existing AnimatedGameSign
