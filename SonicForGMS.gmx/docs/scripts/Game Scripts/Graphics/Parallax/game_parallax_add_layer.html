game_parallax_add_layer

Syntax: 

game_parallax_add_layer(back, left, top, width, height, ox, oy, x_tiled, y_tiled, x_ratio, y_ratio, x_separation, y_separation, x_speed, y_speed);
Argument name Type Description
back real background index
left real background area top-left x-position
top real background area top-left y-position
width real background area width
height real background area height
ox real absolute screen x-position
oy real absolute screen y-position
x_tiled boolean (Optional) whether to tile the background horizontally
y_tiled boolean (Optional) whether to tile the background vertically
x_ratio real (Optional) factor to scroll the x-position against the view's x-position
y_ratio real (Optional) factor to scroll the y-position against the view's y-position
x_separation real (Optional) amount of horizontal space between tiled images
y_separation real (Optional) amount of vertical space between tiled images
x_speed real (Optional) horizontal speed to automatically scroll
y_speed real (Optional) vertical speed to automatically scroll

Returns: (real) instance index of ParallaxLayer (or undefined on error)

Description:

Creates a new parallax background with the given properties
