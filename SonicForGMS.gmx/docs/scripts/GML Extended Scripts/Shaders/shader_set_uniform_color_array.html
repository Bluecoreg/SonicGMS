shader_set_uniform_color_array

Syntax: 

shader_set_uniform_color_array(uniform_id, colors);
Argument name Type Description
uniform_id real shader constant array handle
colors real array of 32-bit color values

Description:

Converts an array of 32-bit color values to a vec3 shader constant array. You must previously have gotten the "handle" of the array using the function shader_get_uniform.
